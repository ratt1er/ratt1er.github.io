这个博客是怎么搭建的-我的第一篇博客

2020-05-04
今天花了半天的时间搭了这个博客，讲一下搭建途中遇到的一些坑顺便也当做第一篇博客来打头阵

1. 安装Node.js

这里推荐用国内的源：传送门
注意，目前(20/05/04)最新的14.x不支持Windows7及以前版本，windows7及以前建议使用12.x版本:传送门

这里的安装没有什么要注意的，一路确定即可。

2.检查是否安装成功检查是否安装成功

打开cmd窗口分别输入
node -v
npm -v
成功的标志就是能分别输出版本号。

3.切换npm的国内源

这里用了阿里云（淘宝）的源

cmd输入

npm config set registry https://registry.npm.taobao.org/
同2，输入

cnpm -v
应该能看到输出的版本号

4.安装Hexo

cmd输入

cnpm install -g hexo-cli

5.建立本机的hexo博客

新建一个文件夹，专门存放博客文件

下文如无特殊说明均在此文件夹内操作

cmd输入
hexo init

因为国内网络环境原因可能会下载非常慢，可以使用我已经下载好的文件
提取码：nmdo

6.本地运行

如果第五步操作成功目录应该有这些东西：

接着输入hexo s
即可完成搭建

浏览器输入localhost:4000即可查看效果
